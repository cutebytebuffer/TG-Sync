# TG-Sync
TG-Sync is a minecraft plugin which allows you to sync totemguard alerts between velocity instances (*Without Redis*)

<img src="alerts.png" width="800" alt="Alerts Screenshot">

## Installation
**To properly install the plugin you'll have to do the following things:**
  after either downloading the built jars or building them from source you'll have to place them in the servers plugin folder (Paper & Velocity)

## Common Issues
 **If you are not reciving any alerts after installing both plugins in their correct server check the following things:**
 - The user has the totemguard.alerts permission on the velocity side by using a plugin such as [LPV](https://luckperms.net/download)
 - The plugin relies on the TotemGuard api so you'll have to have TotemGuard installed alongside TG-Sync (on the paper side)
